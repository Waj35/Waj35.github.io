## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Code editor (VS Code, Sublime Text, etc.) for customization

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Waj35/portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd portfolio
   ```

3. **Open the project**
   - Open `index.html` in your web browser
   - Or use a local development server for better experience

### Using Live Server (Optional)
```bash
# If you have Node.js installed
npx live-server
```

## 🎨 Customization Guide

### Updating Personal Information
Edit the content in `index.html` to reflect your own information:
- Profile details
- Skills and expertise
- Project descriptions
- Contact information

### Changing Colors & Theme
Modify CSS variables in `assets/css/styles.css`:
```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
  --background-color: #your-color;
  --text-color: #your-color;
}
```

### Adding New Projects
1. Locate the projects section in `index.html`
2. Copy an existing project card structure
3. Update with your project details
4. Add project images to `assets/images/projects/`

### Modifying Sections
You can add, remove, or reorder sections by editing the HTML structure while maintaining the responsive design principles.
